Dlp Slicer

October 2022


I created a Slicing Software for DLP and resin printers. Using voxelization method, I extract a grid matrix and transform it to a bitmap image. However, as voxelization is just a logical array representing each pixels, the size of the array adds up and is very ram intensive. Some ram saving strategies are using uint8, sparse array, and chunking.I implemented threading to improve processing speed and to have a responsive gui.
